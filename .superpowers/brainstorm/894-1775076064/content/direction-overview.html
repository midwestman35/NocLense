Phase Rooms

Each workflow phase is a distinct "room" with its own layout, tools, and density. Transitioning between phases uses full-screen morphing animations. The log grid only appears in Investigate. Import is a centered card. Submit is a focused form.

Inspiration
Linear's project views, Figma's mode switching
Animation
Room-to-room morph transitions via anime.js
Key idea
The layout itself communicates where you are in the investigation. No phase stepper needed — the room IS the phase.

Card Workspace

Replace the fixed 3-panel layout with a flexible card canvas. Each investigation tool (Log Stream, AI, Timeline, Evidence, Ticket) is a draggable, resizable card. Cards can be expanded to full-width, collapsed to a header, or rearranged. Think Notion databases meet Miro boards.

Inspiration
Notion blocks, Core.so panels, Arc browser spaces
Animation
Card expand/collapse, drag physics, stagger reveals
Key idea
Users compose their own workspace per investigation. Power users expand the log grid to 80% of screen. Quick-close analysts might just use AI + Ticket cards.

Refined Current

Keep the 3-panel structure but dramatically improve the aesthetics, animation, and phase differentiation. Rounded surfaces, glassmorphism headers, the phase stepper becomes a prominent visual ribbon, and anime.js drives every transition. Think "current layout, but if Apple designed it."

Inspiration
Warp terminal, Raycast, Apple Music
Animation
Smooth panel transitions, phase ribbon glow, micro-interactions everywhere
Key idea
Lowest risk, highest polish. Same mental model for existing users, but everything feels premium. Phase transitions use anime.js timeline to morph the layout subtly between modes.
