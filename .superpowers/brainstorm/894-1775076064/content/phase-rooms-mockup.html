How Phase Rooms work
  • Import Room — centered card, minimal chrome. Focused on getting data in. No log grid, no sidebar.
  • Investigate Room — full workspace: log grid + timeline + filter bar + AI sidebar with tabs. Maximum density for analysis.
  • Submit Room — focused two-card layout: closure note + evidence summary. No log grid. Clean handoff.
  • Transitions — anime.js morphs the entire layout between rooms. Import card expands into the log grid. Sidebar slides in. Submit cards emerge from the AI panel.
  • Back-navigation — click any phase dot in the header to go back. Layout morphs in reverse.
  • The header evolves — Import shows just the logo + phase dots. Investigate adds ticket ID + priority. Submit shows completion state.
